# OCR Demo

This project is an OCR (Optical Character Recognition) demo application built with React Router, React, and DeepInfra's OCR model.

## Setup

1.  **Install Dependencies:**

    ```bash
    pnpm install
    ```

2.  **Environment Variables:**

    Create a `.env` file in the root of the project and add your DeepInfra API key:

    ```
    DEEPINFRA_API_KEY=your_deepinfra_api_key_here
    ```

    Replace `your_deepinfra_api_key_here` with your actual API key from DeepInfra.

3.  **Run Development Server:**

    ```bash
    pnpm dev
    ```

    The application will be available at `http://localhost:5173`.
